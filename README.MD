# 📊 Avaliação de Células

Aplicação simples em **HTML + JavaScript puro** para avaliação prática da saúde de uma célula, baseada em princípios de liderança, relacionamento, organização e espiritualidade.

Funciona **100% offline**, sem dependências externas, podendo ser aberta diretamente no navegador.

---

## 🚀 Objetivo

Este projeto foi criado para ajudar líderes a:

- Avaliar a qualidade da sua célula de forma estruturada  
- Identificar pontos fortes e pontos de melhoria  
- Gerar insights automáticos  
- Planejar ações semanais de melhoria  
- Criar consistência na liderança  

---

## 🧩 Funcionalidades

- ✅ Checklist completo com pontuação (0 a 5)  
- ✅ Cálculo automático por categoria  
- ✅ Score total com classificação (saudável, atenção, risco)  
- ✅ Insights automáticos (pontos fortes e prioridades)  
- ✅ Campo "O que não estou vendo"  
- ✅ Plano semanal de melhoria integrado  
- ✅ Salvamento automático (LocalStorage)  
- ✅ Funciona offline (sem internet)  
- ✅ Opção de impressão / PDF  

---

## 🧠 Estrutura de Avaliação

A avaliação é dividida em 5 pilares:

1. **Vida (Relacionamentos)**
2. **Liderança**
3. **Identidade e Organização**
4. **Ambiente**
5. **Espiritualidade**

Cada item recebe uma nota de **0 a 5**, totalizando até **125 pontos**.

---

## 📊 Classificação

| Pontuação | Status              |
|----------|-------------------|
| 100–125  | ✅ Célula saudável |
| 70–99    | ⚠️ Atenção        |
| < 70     | ❌ Em risco        |
